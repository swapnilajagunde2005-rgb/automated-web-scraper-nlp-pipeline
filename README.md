# 📊 Real-Time Automated Web Scraper & NLP Sentiment Pipeline

An end-to-end data engineering and artificial intelligence application that automates the collection, parsing, and emotional trend tracking of live web strings. This project bridges the gap between backend algorithm execution and customer-facing business logic.



## 🛠️ Tech Stack & Architecture
* **Frontend Dashboard:** Streamlit (UI Architecture & Live Visualization)
* **Data Processing Engine:** Pandas (Matrix Manipulation & Clean Tabular Restructuring)
* **Natural Language Processing:** NLTK VADER (Lexicon & Rule-based Sentiment Intensity Scoring)
* **Web Scraping Infrastructure:** BeautifulSoup4 & Requests (DOM Traversal & Document Parsing)

## 🚀 Key Features
* **Live Web Ingestion:** Safely queries target servers, injects headers to mimic desktop browser agents, and captures raw HTML text segments.
* **Automated NLP Processing:** Maps raw text through a rule-based tokenizer to output exact compound positivity, negativity, and neutrality scores.
* **Interactive Data Grid:** Renders an Excel-style Pandas data matrix directly in the user's web browser, supporting interactive sorting and analytics.
* **Data Exporter:** Features a built-in download engine allowing non-technical stakeholders to export processed metrics straight into a `.csv` file.
